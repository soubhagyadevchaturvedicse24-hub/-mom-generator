## How to Use

1. Open the website
2. Choose **Create Notice** or **Create MOM**
3. Fill in the form (or use pre-filled example data)
4. Click **Generate**
5. **Copy RTF** or **Copy Text** and paste into Microsoft Word
6. Bold formatting is preserved automatically!
