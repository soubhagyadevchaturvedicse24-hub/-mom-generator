### "API key invalid" Error
**Solution:**
1. Go to https://aistudio.google.com/app/apikey
2. Check if key is active
3. Try creating a new key
4. Make sure you copied the entire key (starts with `AIza...`)

### "Quota exceeded" Error
**Rare, but if it happens:**
1. Wait 1 minute (rate limit resets)
2. Or generate fewer MOMs per minute (60 is the limit)

### "API Error" Message
**Check:**
1. Internet connection working?
2. Key copied correctly (no extra spaces)?
3. Try regenerating a new key
4. Fallback: Disable AI, use template generation
